“iS-Commands” is the class we used to group our commands, inside this one, you can insert the main commands. Infinity Slider require only 1 command between Previous, Next, Play, Stop and Dots but you can insert all of them.

“iS-Previous” : If you click an element with this class, the slider will animate the previous-slide-animation

“iS-Next” : if you click an element with this class, the slider will animate the next-slide-animation

“iS-Play” : if you click an element with this class, the slider will start the loop function

“iS-Stop” : if you click an element with this class, the slider will stop the loop function

“iS-Dots” : is a class used to group the “iS-Dot”, you must insert a “iS-Dot” for each “slide-panel” that compose your slider. (e.g. : 5 slide required 5 “iS-Dot”)

We are going to see what “slide-panel” means in the next paragraph.

2.2 How to create your Animations

To create an awesome animation you need to know how the structure was make and what is a slide-panel.

<html lang="en">

<head>

<title>Site Title</title>

</head>

<body>

<div id="your_slider_id" class="iS">

<div class="iS-Commands"> Your Commands </div>

<div class="iS-Content">

<div class="iS-Items" data-loopTime="2000" >

<div class="iS-Item"

        data-effectIn="slideright"

        data-effectInTime="500"

        data-effectInDelay="100" 

        ...more options… >

<img src="images/slide1.jpg" />

</div>

</div>

</div>

</div>

</body>

</html>

We can start analyzing some of the code that compose the animations and the 2 words that we are going to use to explain this part.

Activated : when a slide-panel is activated you can see its elements and its content.

Deactivated : when a slide-panel is deactivated you can not see anything inside that panel.

data-loopTime determinate how much of time the slide will keep activated on the loop function. this value is optional so you can avoid to insert it, as default the slider will use 5000 milliseconds and the minimum is 50ms.

data-effectIn define which effects will be used when the panel is activated, you can insert more effects to create your animation. (you can read more about effects in the next paragraph)

data-effectInTime is used to set the speed of your animations, this value can change from 0 to infinity. If you insert 500, your effects will be animated in 500 milliseconds.

data-effectInDelay , with this value you can create fantastic animations, using a delay on your element, the effects will start after the delay’s value. Change this property from -infinty to +infinity (i. e. : data-effectInDelay = “-200” or data-effectInDelay = “700”)

We used effectIn to define what happens when your panel is activated, now we can use the effectOut to define the “deactivation” effects.

<div class="iS-Item”

        data-effectIn="slideright"  data-effectInTime="500"  data-effectInDelay="100"

        data-effectOut="slideleft" data-effectOutTime="500" data-effectOutDelay="0" >

<img src="images/slide1.jpg" />

</div>

data-effectOut to define which effects used when your panel is deactivated

data-effectOutTime determinate the speed of the effectOut animation.

data-effectOutDelay is the delay for the effectOut animation.

Likewise the effectIn and the effectOut there are some optional data, you can insert it to create a fully configured slider.

<div class="iS-Item”

data-effectIn="slideright"  data-effectInTime="500"  data-effectInDelay="100"

data-effectOut="slideleft" data-effectOutTime="500" data-effectOutDelay="0"

data-effectInBack="slidetop"  data-effectInTimeBack="500"  data-effectInDelayBack="100"

data-effectOutBack="zoomin" data-effectOutTimeBack="500" data-effectOutDelayBack="0" >

<img src="images/slide1.jpg" />

</div>

With these data you can determinate which animations the elements have to do when they return active clicking the “previous button”, obviously if you don’t insert these data, the slider will animate the elements using the normal effectIn and effectOut data.

2.3 Which effects can you insert

Before to start, you have to know 3 rules about the effects.

1) All effects can be combined and inserted as effectIn, effectOut, effectInBack and effectOutBack. To combine an effect you have to insert it with a blank space.

data-effectIn="slideright slidetop zoomin"

2) The order of effects can generate different animations.

3) When you insert an effect as effectIn or effectInBack you have to think about “effect from”

(example : effectIn=”slideright” means “slide from right”), when you insert an effect as effectOut or effectOutBack you have to think about “effect to” (example : effectOut=”slideright” means “slide to right”

Normal Effects

Slide

You can animate your elements from/to Left, Top, Bottom and Right or combine two animations (like top + right) to create a diagonal slide effect.

There are 2 types of slide animations:

1) Offfset Slide: this effect can be called inserting “slideleft” , “slideright” , “slidetop”, “slidebottom” as data-effect value to obtain a good animation combine it with “fadein”

2) Screen Slide: works with slider’s width or height, every element with this effect will be animate from/to out of slider’s sizes. Use “slideleftscreen”, “sliderightscreen”, “slidetopscreen”, “slidebottomscreen” as data-effect value

Fade

A simply opacity effect, perfect to create a clear animation or be combined to develop an special effect. Use “fadein” as data-effect value

Zoom

A CSS3 effect that allows you to insert on your slider an incredible animation of expansion or restriction. Use “zoomin” or “zoomout” (zoomout+fadein to create an awesome animation)

Flip

Rotate your elements on your slide will be really easy with our Flip effects, From/to “flipeft”, “fliptop”, “flipbottom” or “flipright” ready to surprise your customers

Spin

To complete the collection of Flip effects, we have inserted also an Spin effect to provide you an animation for every dimension(x, y, z). Use “spinleft” or “spinright” (combine it with “fadein” to get a good animation)

None

At the end we got also the “none” effect, used (rarely) when you need an element without any effect or animation

                ..more..

        )

2.5 How to choose your Layouts and Colors

Inside Infinity Slider you will find 3 starter layouts to customize your slider.

skin-dots.html

skin-icons.html

skin-thumbnails.html

Each html file contains the same content of index.html but it’s ready to be used with the chosen skin.

To use a skin, the html file change 2 parts of its code:

-the skin css on the header

<link rel="stylesheet" href="skins/skin-dots.css" type="text/css" />

-the skin class on the slider

<div id="slide01" class="iS iS-SkinDots">...</div>

You can easily change the skin-dots.css with skin-icons.css or skin-thumbnails.css

and the iS-SkinDots class with the iS-SkinIcons or iS-SkinThumbnails

There is another class used to get a fullscreen slider, you can add the class iS-Fullscreen  to the slider and the fullscreen.js to the page

<div id="slide01" class="iS iS-SkinDots iS-Fullscreen">...</div>

<script type="text/javascript" src="js/fullscreen.js"></script>

To change the colors, you have to open the file css inside the skin folder and modify arrows, buttons, bullets and every element you wish.

ps: the thumbnails skin require also the thumbnails-horizontal.js

2.6 What is the Loader?

The loader is a new function that allows you to insert a preloader on your slider or webpage. Insert the loader is really easy.

You need to add the following code on your page:

On the head:

<link rel="stylesheet" href="loading/loading-bars.css" type="text/css" media="all" />

On the bottom of the page:

<script type="text/javascript" src="js/loader.js"></script>

And the Main Code:

<div class="iS-Loadingbox">

      <p class="iS-Loadingboxtitle">Loading</p>

      <div class="iS-Loadingboxbar"></div>

      <div class="iS-Loadingboxbar"></div>

      <div class="iS-Loadingboxbar"></div>

      <div class="iS-Loadingboxbar"></div>

</div>

 If you insert the main code inside the slider, the loader will be used ONLY over it.

<div id="slide01" class="iS">        

<div class="iS-Loading">...</div>

…

</div>

Otherwise if you insert it outside the slider, the loader will be available for ALL the page

<div class="iS-Loading">...</div>

<div id="slide01" class="iS">        

…

</div>

At the moment there is only 1 loader for Infinity Slider but, as the skins, we’ll add more in few versions.

2.7 Are there additional features that i need to know?

- Internet Explorer 9 naturally is not compatible with CSS3 animations so we have developed an automatic system that switch the CSS3 animations with the jQuery animations that are supported by the browser.

To active this function you have to insert the following code in your web page

<!--[if IE 9]><div class="iS-IE9"></div><![endif]-->

- Autocheck errors it make an automatic check of the code so if you do some error like write “slidelefft” the system will tip you that the effect does not exist.

- Additional Setup, with these options you can configure special classes and features to your slider

To insert these options you have to add the code and the value to the slider function:

<script>

$(document).ready(function() {

        infinitySlider(

                infinitySliderId = your_slider_id,

                ...

                infinitySliderDotActiveClass = 'iS-Dotactive',

                infinitySliderPlayButtonActiveClass ='iS-Playactive',

                infinitySliderStopButtonActiveClass ='iS-Stopactive',

                infinitySliderLoopIndicator ='iS-Loopline',

                infinitySliderAutoStartLoop = false,

                    infinitySliderKeyboardNavigation = true,

infinitySliderTouchNavigation = 'mobile’,

infinitySliderStarterSlide = 1

        )

});

</script>

</body>

infinitySliderDotActiveClass is a special class that a normal dot (iS-Dot) get when is clicked

(example : infinitySliderDotActiveClass = 'iS-Dotactive' if you click on a <div class=”iS-Dot”> it will become <div class=”iS-Dot iS-Dotactive”>, use it to mark the current slide panel.)

infinitySliderPlayButtonActiveClass is a special class that the “iS-Play” get when is clicked (like “iS-Dot”)

infinitySliderStopButtonActiveClass  is a special class that the “iS-Stop” get when is clicked (like “iS-Dot” and “iS-Play”)

Tip: when “iS-Play” is activated, “iS-Stop” is not and vice versa

infinitySliderLoopIndicator is a special class used by the loopline, when the slider loop is active, this element will get the class “iS-Loopactive”.

infinitySliderAutoStartLoop with this data, you can choose if activate the loop function at the load of the page with “true”

There is a class “iS-Hoverstop” that allows you to stop the playing slider when your cursor is over the element with that class., you can use on the slider or on an element of it

(example: <div class=”iS-Item iS-Hoverstop” …>Hover me to stop the slide</div>)

infinitySliderKeyboardNavigation You can activate the Keyboard Navigation choosing “true” as this value, with this option left and right arrows on your keyboard will be used to animate the slider

infinitySliderTouchNavigation Use this data to choose the touch screen navigation that you need between: desktop, mobile or both

Note: If you are not using infinitySliderPreviousButtonClass or infinitySliderNextButtonClass  KeyboardNavigation and TouchNavigation functions will not works

infinitySliderStarterSlide you can set this value to choose the starter slider (example: if you have 5 slide panel on you slider, you can choose between 1-5) The slider will load the chosen slide on the load of the page
