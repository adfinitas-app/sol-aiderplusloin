Changelog 1.2 - March/31/2014

New:

Fullscreen options for skins, now you can insert a fullscreen option for you slide

Touchscreen compatibility, you can now set up your touchscreen commands with mobile, desktop or both

Starter Slider is now available, choose which slide will show first when the page is loaded

Fixed:

TextHole now can be resized when the slider is not visible

Documentation minor examples bugs

Documentation "none effect" missing

Firstload Check now works properly

Changelog 1.1 - March/17/2014

New:

Blur Variable Value: is now possible, you can choose the radius of blur

iS-Hoverstop class is now available, you can add this class to an element to stop the “autoplay slide”. check the documentation to read more.

Loader Function is a new function that allows you to cover the slider on loading fo page and images. check the documentation to read more.

Fixed:

TextHole is now selectable properly on Internet Explorer and Mozilla Firefox

Skins improved with a better responsive and removed minor bugs

Autoplay Slide 

will start when everything is loaded

will stop if there is a video with VideoAutoPlay function

Autoplay Video

will start when the video is visible on the slider.

Blur and Grayscale effects are now fully compatible with Internet Explorer 9 and can be combine, using canvas instead of svg this function will be faster, better and more compatible.
